# Core

## Overview

Use this skill as the default baseline for software delivery workflows.

## What it adds

- Core pipelines for feature delivery, quick fixes, and PoC work.
- Core agents for planning, architecture, and review handoffs.
- Core prompts for planning, implementation flow, and final review before push.

## When to use

- Any project that needs plan-first execution and consistent review gates.
- Any task where TDD, security checks, and structured delivery are expected.
