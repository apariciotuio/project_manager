# QA Playwright

## Overview

Use this skill for E2E and acceptance test automation with Playwright.

## What it adds

- QA automation standards for page objects, test data, and stability.
- Agents: `qa-automation-engineer`, `qa-automation-engineer-experimental`.
- Prompts: `plan-qa-tests`, `develop-qa-tests`.

## When to use

- Creating new E2E coverage from acceptance criteria.
- Maintaining brittle tests after UI/API changes.

## Related prompts and agents

- Plan with `plan-qa-tests`.
- Implement and maintain with `develop-qa-tests`.
