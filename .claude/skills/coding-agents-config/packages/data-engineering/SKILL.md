# Data Engineering

## Overview

Use this skill for Databricks data pipelines using medallion architecture.

## What it adds

- Data engineering standards: bronze/silver/gold, Delta Lake, DLT, Unity Catalog.
- Agent: `data-engineer`.
- Prompts: `plan-data-pipeline`, `develop-data-pipeline`, `aidata-tasks`.

## When to use

- Building or extending ingestion/transformation pipelines.
- Adding data quality and governance controls.

## Related prompts and agents

- Plan with `plan-data-pipeline`.
- Implement with `develop-data-pipeline`.
