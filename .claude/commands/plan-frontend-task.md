# Plan Frontend Task

Create a detailed frontend implementation plan.

## Input

Task ID as argument. Expects `tasks/<TASK-ID>/` with specs and design.

## Process

1. Read existing specs and design from `tasks/<TASK-ID>/`
2. Create `tasks/<TASK-ID>/plan-frontend.md` with step-by-step implementation plan
3. Plan components, services, routing, state management
4. Each step references WHEN/THEN/AND scenarios
5. Include accessibility and security considerations

## Output

Saves plan to `tasks/<TASK-ID>/plan-frontend.md`. Blueprint for `develop-frontend`.
