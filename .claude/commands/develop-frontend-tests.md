# Develop Frontend Tests

Write tests for existing untested frontend code.

## Process

1. Analyze existing frontend code for untested paths
2. Write failing tests — behavior over implementation
3. Triangulate with multiple test cases
4. Report: tests written, coverage impact, testability concerns

## Rules

- NEVER write production code
- Fakes over mocks
- Test behavior, not implementation details
