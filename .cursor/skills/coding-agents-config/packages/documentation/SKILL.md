# Documentation

## Overview

Use this skill to keep technical documentation aligned with implementation changes.

## What it adds

- Documentation standards for READMEs, changelogs, API docs, and ADRs.
- Prompt: `update-docs`.

## When to use

- After feature delivery, behavior changes, or API updates.
- During release preparation and maintenance passes.

## Related prompts and agents

- Use `update-docs` to apply the documentation update workflow.
