# ML Engineering

## Overview

Use this skill for ML model lifecycle work on Databricks and MLflow.

## What it adds

- ML engineering standards for feature engineering, tracking, registry, and monitoring.
- Agent: `ml-engineer`.
- Prompts: `plan-ml-model`, `develop-ml-model`.

## When to use

- Designing and implementing predictive model workflows.
- Operationalizing model training, evaluation, and release.

## Related prompts and agents

- Plan with `plan-ml-model`.
- Execute with `develop-ml-model`.
