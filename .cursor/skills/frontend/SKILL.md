# Frontend

## Overview

Use this skill when building or modifying frontend behavior and user interfaces.

## What it adds

- Agent: `frontend-developer`.
- Prompts: `plan-frontend-task`, `develop-frontend`.
- Works with core and security/testing packages for full pipeline enforcement.

## When to use

- New UI features, component refactors, and interaction changes.
- Frontend tasks requiring plan-first execution.

## Related prompts and agents

- Start with `plan-frontend-task` for feature work.
- Implement with `develop-frontend`.
