# Test

Execute testing workflow — run tests, analyze failures, report results and coverage.

## Process

1. Identify test scope (backend, frontend, or all)
2. Run test suite
3. Analyze failures — root cause, not just symptoms
4. Report: pass/fail counts, coverage, failing tests with diagnosis

## Rules

- NEVER write production code
- Report failures with actionable diagnosis
- Suggest fixes but don't implement them
